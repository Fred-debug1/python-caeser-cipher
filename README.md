# Caesar Cipher (Python)

A simple Caesar cipher encoder and decoder written in Python.

## Features
- Encode text using a Caesar cipher
- Decode Caesar-ciphered text
- Handles spaces correctly
- Case-insensitive input

## How It Works
Each letter is shifted forward or backward in the alphabet by a user-defined number.
